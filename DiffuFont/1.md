**目标**：学一个 `StyleEncoder + TokenLayer`  
输入：同一字体的 reference 集合（每步 8 张，来自 16 桶）  
输出：`style_tokens ∈ [B_style, T, D]`（推荐 `T=8, D=256`）  
训练方式：对比学习 + 稳定性 + token 分工

你当前配置建议：

* `B_style = 32`
* `N_ref = 8`
* `views = 2`（R1/R2）
* 图像：256 原图 → 增强 → resize 到 128 喂 encoder

## C. 预训练采样策略（每个 step）

对一个 batch（32 个字体）中的每个字体 `F`：

### C1. 先按桶采 8 个 reference（强制不同桶）

* 从 16 桶里随机选 8 个**不重复桶**
* 每个桶随机取 1 个字（桶内轮换/洗牌更好）
* 得到 `R`（8 refs）

### C2. 两视图（R1 / R2）

同一个字体再独立采一次得到 `R2`：

* `R1 = sample_bucket_8(F)`
* `R2 = sample_bucket_8(F)`

（可以允许少量字重叠，但不建议完全相同）

---

## D. Dropout 策略（你要的参数在这里）

### D1. Reference dropout（必须）

目的：防止模型依赖某个固定组合、提升对 reference 数量变化的鲁棒性。

* `p_ref_drop = 0.25`
* 从 8 个里随机 drop，最终 **保留 6 左右**
* **最少保留 4**（低于 4 就重采/补齐）

> 你不想固定 reference，这个是最重要的“去捷径”手段。

### D2. Token dropout（预训练可选，轻量）

预训练阶段可以不开；如果开就很小：

* `p_token_drop = 0.05 ~ 0.10`
* 8 tokens 随机 drop 0~1 个

（更推荐把 token dropout 留到 diffusion 阶段做）

---

## E. 图像增强（256 → 128）

每张 reference 图做独立增强，**每次都不同**：

**顺序**（推荐）：

1. `RandomResizedCrop`：scale `0.6~0.9`，ratio `1.0`
2. `RandomMask/Cutout`：面积比例 `0.2~0.4`
3. `RandomAffine`：rotate `±5°`，translate `±5%`，scale `0.9~1.1`
4. resize → `128×128`

---

## F. 模型结构（简单但好用）

### F1. StyleEncoder（CNN）

* 输入：`[B_style*views*N_ref_eff, 1, 128, 128]`
* 输出 feature map：`[*, C, h, w]`（推荐 `C=256, h=w=8`）

### F2. Patch tokens

* flatten：`[*, P, C]`，`P=h*w=64`

### F3. TokenLayer（Set Aggregation）

把同一个字体的一组 reference 的 patch tokens 合并为集合：

* `[B_style, N_ref_eff*P, C]`

用 `T=8` 个 learnable queries 做 cross-attention 聚合：

* 输出 `style_tokens: [B_style, 8, 256]`

---

## G. Loss 设计（不含 L_char_adv）

你会得到两视图输出：

* `tokens1, tokens2: [B_style, T, D]`

为对比学习取一个向量（只用于 loss）：

* `z1 = mean(tokens1, dim=token)` → `[B, D]`
* `z2 = mean(tokens2, dim=token)` → `[B, D]`  
    并对 `z` 做 L2 normalize。

### G1. 主损失：InfoNCE / SupCon（必须）

* 正样本：同字体 `(z1_i, z2_i)`
* 负样本：batch 内其他字体
* 温度：`τ = 0.07`

### G2. 风格一致性：Consistency（强烈推荐）

解决你之后最在意的 **style drift**：

* `L_cons = 1 - cosine(z1, z2)`（或 MSE）
* 权重：`λ_cons = 0.2`

### G3. Token 分工：Token Diversity（推荐）

防止 token collapse：

* 对 tokens 做 normalize
* 计算 token-token 相似矩阵 `S`
* 取 off-diagonal 的 `S^2` 平均
* 权重：`λ_div = 0.01`

### 最终 loss（推荐默认）

Plain textL = L_InfoNCE  
  + 0.2  * L_cons  
  + 0.01 * L_div

## H. 预训练完成后的输出

你保存：

* `StyleEncoder` 权重
* `TokenLayer` 权重（learnable queries + attention）

推理/下游使用时输出：

* `style_tokens: [B, 8, 256]`（不做 mean）

## I. 接入 diffusion 训练时的 dropout（建议）

当你用预训练模块去 condition UNet 时：

* 仍然按桶采样 `N_ref=8~16`（推荐训练 diffusion 用 8/12/16 随机）
* **reference dropout**：`p_ref_drop = 0.10~0.20`（比预训练更轻）
* **token dropout**：`p_token_drop = 0.10`
